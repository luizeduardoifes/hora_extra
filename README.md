**Resumo**
Este sistema foi feito para registrar e calcular quantas horas extras uma pessoa tem, ajudando a ter uma noção clara do saldo de horas em casa e no trabalho. Ele também serve como um exercício prático para aprender lógica de programação, manipulação de dados simples e organização de informações em arquivos de texto.

O objetivo é facilitar o acompanhamento das horas extras no dia a dia, mantendo o controle de entradas, saídas e totais, além de reforçar conceitos básicos de programação de forma aplicada.
